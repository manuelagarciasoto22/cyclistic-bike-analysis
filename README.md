## Análisis de Uso del Sistema de Bicicletas Cyclistic
## Descripción del Proyecto

Este proyecto tiene como objetivo analizar el comportamiento de los usuarios del sistema de bicicletas Cyclistic mediante el uso de herramientas de inteligencia de negocios.

Se utilizaron datos de viajes del primer trimestre de 2020 para identificar patrones de uso entre usuarios miembros y ocasionales.

## Herramientas utilizadas

- Power BI
- Análisis de datos
- Limpieza y transformación de datos
- Visualización de datos

## Proceso realizado

Para el desarrollo del proyecto se realizaron las siguientes etapas:

- Preparación y limpieza de los datos
- Transformación de variables como duración del viaje y día de la semana
- Análisis descriptivo de los datos
- Creación de visualizaciones y dashboard en Power BI

## Principales hallazgos

El análisis permitió identificar que:

- Los usuarios ocasionales realizan viajes de mayor duración.
- Los miembros utilizan el sistema con mayor frecuencia.
- El uso del sistema por parte de usuarios ocasionales aumenta durante los fines de semana.

Estos resultados permiten comprender las diferencias en el comportamiento de los usuarios.
